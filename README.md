🚀 Projet Shadow Client — Analyse de Données (Portfolio Data Analyst)
Python Pandas Visualization Status

📊 Aperçu du projet
Shadow Client est un projet d’analyse de données conçu pour démontrer des compétences professionnelles en exploration, nettoyage, visualisation et extraction d’insights à partir de données commerciales.
Objectif : transformer des données brutes en informations exploitables pour la prise de décision business.
Ce projet fait partie d’un portfolio destiné au freelancing et à la présentation professionnelle sur GitHub et LinkedIn

🎯 Objectifs
•	Comprendre la structure d’un dataset commercial
•	Standardiser les colonnes et préparer les données
•	Identifier les catégories les plus rentables
•	Détecter les anomalies (profits négatifs)
•	Produire des visualisations claires et interprétables
________________________________________

🔎 Étapes de l’analyse
1️⃣ Chargement et préparation
•	Importation des bibliothèques Python
•	Chargement du dataset CSV
•	Standardisation automatique des colonnes
2️⃣ Exploration des données
•	Aperçu des premières lignes
•	Informations générales (info())
•	Statistiques descriptives (describe())
•	Analyse des valeurs manquantes
3️⃣ Visualisations
•	Histogrammes des variables numériques
•	Heatmap de corrélation
•	Analyse des catégories principales
4️⃣ Analyse avancée
•	Calcul du profit total
•	Analyse par catégorie
•	Détection des profits négatifs
•	Analyse des corrélations importantes
